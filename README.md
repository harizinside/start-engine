# Oak Boilerplate
This is a boilerplate for building web applications using the Oak framework in Deno. It provides a basic structure for setting up routes, middleware, and handling requests.

```
oak-api/
├── deno.json
├── deno.lock
└── src
    ├── index.ts
    ├── app.ts
    ├── routes
    │   └── index.ts
    └── controllers
        └── health.controller.ts
```
